# Proyecto de bebidas

Herramientas y tecnologías utilizadas:

+ React + Vite
+ Firestore Database
+ React Router DOM
+ Context API
+ Vercel

## Guía de instalación

1. Instalar dependencias 

```
npm install
```

2. Levantar proyecto

```shell script
npm run dev
```

Link del deploy: [Deploy Vercel](https://crea-tu-landing-matayoshi.vercel.app/)
